# Project Refactoring: Scripting World 🌍

Welcome to the newly refactored **NIT KKR Marketplace** project. The entire codebase has been organized into a professional directory structure, separating structural HTML, stylistic CSS, and interactive JavaScript logic.

## 📂 Project Structure

```text
projectS/
├── html/             # Structural HTML components
│   ├── index.html     # Homepage
│   ├── browse.html    # Item Catalog
│   ├── dashboard.html # User Dashboard
│   ├── auth.html      # Login & Signup
│   ├── chat.html      # Messaging Interface
│   └── admin.html     # Administrator Panel
├── css/              # Visual Styling
│   ├── index.css
│   ├── browse.css
│   ├── dashboard.css
│   ├── auth.css
│   ├── chat.css
│   └── admin.css
└── js/               # Interactive Logic (Scripting)
    ├── index.js
    ├── browse.js
    ├── dashboard.js
    ├── auth.js
    ├── chat.js
    └── admin.js
```

## 🛠 Key Features Implemented

1.  **Professional Scripting**: All JavaScript is now modularized in the `js/` folder, allowing for easier debugging and feature expansion.
2.  **Thematic Consistency**: Dark and Light modes are preserved and synchronized across all pages.
3.  **Quick Links Integration**: Navigation menus have been updated to include consistent links to Home, Browse, Dashboard, and Messages.
4.  **Copyright 2026**: All footers have been updated with the current project year.
5.  **Clean Code**: Inline styles and scripts have been removed to follow modern web development best practices.

## 🚀 How to Run

Simply open any of the files in the `html/` folder using your browser. The relative paths `../css/` and `../js/` will correctly load the associated styles and scripts.
